# SubscriptionSentry Design Brief

## Visual Identity

### Color Palette
- **Primary Color**: Blue (#007AFF) - Represents trust and professionalism.
- **Secondary Color**: Green (#34C759) - Symbolizes efficiency and success.
- **Accent Color**: Orange (#FF9500) - Used for alerts and notifications to draw attention.
- **Background Color**: Light Gray (#F2F2F7) - Provides a neutral backdrop to ensure content prominence.
- **Text Color**: Dark Gray (#1C1C1E) - Ensures readability against the light background.

### Mood/Tone
The design should convey a sense of clarity, efficiency, and trustworthiness. It should feel modern and professional, appealing to tech-savvy individuals and digital artists alike.

## Typography

### Fonts
- **Heading Font**: Montserrat (Google Font)
  - Sizes: 24px (H1), 20px (H2), 18px (H3)
  - Weights: Bold (700) for headings to ensure clear hierarchy and emphasis.
  
- **Body Font**: Roboto (Google Font)
  - Sizes: 16px for body text, 14px for secondary text
  - Weights: Regular (400) for standard text, Medium (500) for emphasis.

## Component Library

1. **Navigation Bar**: 
   - Style: Bottom navigation with icons and text labels.
   - Icons: Home, Scan, Dashboard, Settings.

2. **Buttons**:
   - Primary Button: Filled with Primary Color (Blue), rounded corners, white text.
   - Secondary Button: Outline with Primary Color (Blue), rounded corners, blue text.

3. **Input Fields**:
   - Style: Rounded corners, light background (Light Gray), dark text (Dark Gray).

4. **Cards**:
   - Style: Elevated with shadow, rounded corners, used for displaying subscription details.

5. **Alerts**:
   - Style: Banner with Accent Color (Orange) for redundancy notifications.

6. **Charts**:
   - Style: Line and bar charts with Primary and Secondary Colors to represent different data points.

## Key Screen Layouts

1. **Home Screen**:
   - Top Section: Welcome message and quick access to recent scans.
   - Middle Section: Button to start a new scan.
   - Bottom Navigation: Access to Dashboard, Scan, and Settings.

2. **Scan Screen**:
   - Camera Viewfinder: Full-screen with capture button.
   - Instructions Overlay: Brief guide on how to take a photo.

3. **Dashboard Screen**:
   - Top Section: Monthly spending overview with a line chart.
   - Middle Section: List of categorized subscriptions.
   - Bottom Section: ROI Calculator with input fields for usage frequency.

4. **Alerts Screen**:
   - List view of redundancy alerts with action buttons to manage subscriptions.

## Responsive Strategy

- **Mobile**: Primary focus, designed for iPhone screens.
- **Tablet**: Ensure components scale appropriately, use split-view for certain sections.
- **Desktop**: Not applicable as the app is iOS-only.

## Micro-interactions

- **Button Press**: Scale slightly on press for tactile feedback.
- **Alert Notifications**: Slide in from the top with a bounce effect.
- **Chart Animations**: Smooth transitions when data updates or changes.

## Accessibility

- **Contrast**: Ensure text and interactive elements have a contrast ratio of at least 4.5:1.
- **Text Resizing**: Support dynamic type adjustments for users with vision impairments.
- **VoiceOver**: Ensure all interactive elements are labeled appropriately for screen readers.
- **Touch Targets**: Minimum 44x44px touch area for all interactive elements to accommodate users with motor impairments.

This design brief aims to create a cohesive and user-friendly experience for SubscriptionSentry, ensuring both functionality and aesthetic appeal align with the target audience's needs.
